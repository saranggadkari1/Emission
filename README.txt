EPATrac v1.0
EPA Emission Compliance Tracking Tool
Pilot Systems International LLC
============================================================

QUICK START
-----------
1. Run GET-NODE.bat once to download portable Node.js (Windows).
2. Double-click EPATrac.bat to launch.
3. Your browser opens automatically at http://127.0.0.1:49152
4. Complete the 8-question classification wizard to determine which
   regulations apply to your vehicle/engine program.
5. Use the Track tab to manage compliance status, criteria, and assignments.

REGULATIONS COVERED
-------------------
- 40 CFR Part 86    — Control of Emissions from New and In-Use Highway
                      Vehicles and Engines (Subparts A, B, E, G, K, L, N, S)
- 40 CFR Part 1036  — Control of Emissions from New and In-Use Heavy-Duty
                      Highway Engines (all subparts)
- 40 CFR Part 1037  — Control of Emissions from New Heavy-Duty Motor Vehicles
                      (all subparts)
- 40 CFR Part 1065  — Engine-Testing Procedures (all subparts)

FEATURES (v1.0)
---------------
- 8-question EPA Phase 2 HD classification wizard
  (GVWR, vehicle type, engine cycle, HDE class, model year, fuel,
   completeness, special provisions)
- ~95+ trackable regulatory sections with applicability gating
- Atomic acceptance criteria across all sections (Pass/Fail/In Progress/N/A)
  Types: Numerical Limits, Test Procedures, Certification/Documentation, ABT
- Auto Red/Yellow/Green roll-up from criteria
- 28 EPA certification and reporting forms with applicability highlighting
- 12 filing deadlines sorted by urgency
- Named users directory with role tracking and autocomplete
- Owner Workload view: List + Gantt chart (group by user or section)
- Compliance summary report (Markdown / HTML / Print)
- CSV export (one row per criterion) and JSON backup/import
- Dark mode default; light mode via system preference
- Pilot Systems crimson branding

DATA LOCATION
-------------
Your data is saved to:  epatrac-data.json  (same folder as EPATrac.bat)

Back up this file regularly using the JSON Backup button in the app.
To restore: use the Import button and select your backup file.
