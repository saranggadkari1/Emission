FMVSSTrac — FMVSS Compliance Tracker
Pilot Systems International LLC
=====================================

QUICK START
-----------
1. If you don't have Node.js installed: double-click GET-NODE.bat (one-time setup)
2. Double-click FMVSSTrac.bat to launch
3. Your browser will open automatically at http://127.0.0.1:49152

HOW TO USE
----------
1. CLASSIFY: Answer the 11 questions in the wizard to identify which FMVSS 
   standards apply to your vehicle program.

2. TRACK: For each applicable standard, set RYG status, assign owners,
   set due dates, track % complete, and add subtasks.

3. DASHBOARD: See a high-level summary of all sections by status.

4. EXPORT: Download CSV or JSON backup from the toolbar.

DATA
----
All data is saved automatically to fmvsstrac-data.json in this folder.
Back up that file regularly.

STANDARDS COVERED
-----------------
49 CFR Part 571  — Federal Motor Vehicle Safety Standards (FMVSS)
49 CFR Part 567  — Certification requirements
49 CFR Part 568  — Vehicles built in two or more stages

SUPPORT
-------
To update for regulatory changes, see the FMVSSTrac Replication Guide.
